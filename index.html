I am
Agnes Iizuka

My name is Agnes Yoko Iizuka, a twenty year old fashion designer from Mississauga. I am studying fashion design at Ryerson University.

I am passionate about designing beautiful clothing from start to finish. My design process takes a project all the way from an illustration to production. My projects journey through trial and error until it reaches perfection. Currently I am exploring to discover what part of fashion interests me most.

Drape

3rd Year Project

A draping exercise: use plaid reversible fabric to create a unique design on the mannequin. This design plays with pleating and plaid matching; since it is symmetrical it is draped only on one side.

Evening Wear

3rd Year Project

A design inspired by the abstract theme “Surreptitious”. The dress was first designed through illustration and draping. Before completion, muslin of similar materials was created; the final dress is made of silk taffeta and organza overlay. The dress has a simple silhouette, a high-low skirt with a princess seam strapless bodice.

Hat

2nd Year Project

Emma Watson’s bunny fascinators featured in editorial shoots inspired the “Bunny Hat”. The hat was created using steam and wooden hat blocks in order to form the head shape. The actual bow, was cut, molded and sewn by hand.

Hat

2nd Year Project

The first attempt at millinery was this hat designed inspired by the boater hat. Although less structured the hat was created through the cut and sew method. The teal ribbon was braided and sewn in place by hand.

Jacket

3rd Year Project

The jacket was part of a women’s coordinated outfit, featuring the plaid jacket and red-orange shorts. The project was inspired by the colouring and structure in the painting “The Arch of Septimius Severus” by Luigi Bazzani.

The jacket was designed through flat patternmaking and plaid matching.

Plus Size

1st Year Project

The first construction project created was made using plus-size blocks for a daywear dress. The dress is inspired by the “Rose Garden” theme, featured in the scalloped collar and colouring. The design was required to have pockets, sleeves and 3 alterations (godets, gathers, yokes).

Recreate

3rd Year Project

This project was created with fellow design student Jacqueline Au. The project was based on original design by Roland Mouret. The back was redesigned, but the front is pattern drafted based on the original front. Although partnered while drafting the design, the dresses were created individually using velvet back cotton.

Watercolour

2nd Year Project

An illustration based on drapes done by classmates in the same class. The pose is based on nude life drawings done in class. The illustration is created using watercolour pencils, markers and a marker blender.

Watercolour

2nd Year Project

This set of illustrations is a project to present the same eveningwear design in different arrangements. The illustrations are shown as croquis, crop and editorial. All illustrations were created through watercolour.

Mixed Media

2nd Year Project

This project is created using the exploration of media : watercolour pencils, marker blender, and gouache. The illustration was referencing unique interpretations of drapes done by other students in the class.

Frottage

2nd Year Project

A technique of rubbing coloured pencils atop various surfaces to create texture was used to create these pieces. They were created in 2 different colour ways.
